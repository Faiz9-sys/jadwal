🔑 GitHub Token
📋 كيف تحصل على Token؟
1. اذهب لـ github.com/settings/tokens/new
2. اكتب اسم مثل "jadwal-app"
3. حدّد مدة انتهاء (90 يوماً مثلاً) وجدّده دورياً — تجنّب "No expiration"
4. الأفضل: Fine-grained Token مقيّد بمستودع التطبيق فقط (صلاحية Contents: Read & Write)
⚠️ التوكن يُحفظ في متصفحك فقط — لا تمنحه صلاحيات أوسع من اللازم
